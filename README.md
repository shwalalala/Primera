## Primera

### An AI-assisted pregnancy support app using smartwatch data, voice input, and personalized goals to improve tracking, engagement, and maternal care.

## Technology Stack

- **Frontend**: Jetpack Compose (UI Toolkit)
- **Backend**: Firebase, Firestore
- **Dependencies**: Health Connect API, Smartwatch API
- **Integration**: Speech-to-Text (RecognitionService), Kotlin Coroutines
- **Language**: Kotlin

## General Objectives

- **General Objective 1**: Improve Maternal Health Entry Through Smartwatch-Based Physiological Data Integration
- **General Objective 2**: Improve Maternal Health Data Entry Through AI-Assisted Speech-to-Text Logging
- **General Objective 3**: Enhance Maternal Engagement Through Personalized Wellness Goals, Progress Tracking, and Supportive Reinforcement
